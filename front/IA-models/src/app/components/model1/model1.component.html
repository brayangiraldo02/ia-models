Reconocimiento de números.

Este modelo es capaz de reconocer números escritos a mano, dibujados en el canvas que está en la parte de abajo.

Las librerías y pasos para el entrenamiento han sido los siguientes:

  • Se ha utilizado este video como apoyo para realizar la red neuronal
  • Para el entrenamiento de esta red neuronal, se ha utilizado la librería de código abierto TensorFlow.
  • Se ha utilizado un Dataset Mnist que contiene 60.000 imágenes de números escritos a mano de 28x28 pixeles, como se puede visualizar a la derecha.
  • Se ha utilizado una red neuronal convolucional (CNN) para el entrenamiento de este modelo.
  • Se han asignado 5 capas, una con profundidad de 32, otras dos de 64 y otras dos de 128.
  • Nuestro código fuente de este modelo y esta página de presentación, se encuentra en este repositorio de GitHub.
