Reconocimiento de letras.

Este modelo es capaz de reconocer letras escritos a mano, dibujados en el canvas que está en la parte de abajo. Además, reconoce los números mostrados en la anterior versión.

Las librerías y pasos para el entrenamiento han sido los siguientes:

  • Para el entrenamiento de esta red neuronal, se ha utilizado la librería de código abierto TensorFlow.
  • Se ha utilizado un Dataset Emnist que contiene 88.000 imágenes de letras escritas a mano de 28x28 pixeles, como se puede visualizar a la derecha.
  • Se ha utilizado una red neuronal convolucional (CNN) para el entrenamiento de este modelo.
  • Se han asignado 5 capas, una con profundidad de 32, otras dos de 64 y otras dos de 128.
